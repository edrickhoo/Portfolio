Games List App

This project was made with React, React-Router and styled with TailWindCSS. The GamesList app is able to search through an API to find games matching the certain titles and display those games. Each game is able to be favourited and filtered by major platform categories. Favourited games can also be filtered and are displayed the favourites section. On refresh local storage is used to retrieve the favourites data.

Built with: React, TailwindCSS and React Router DOM

Google Books List App

This project will asynchronously fetch data from google books api. The data return will be dynamically rendered depending on the data from the API. The user can enter specific titles to search for through the input box. Users can click more info and a modal will appear displaying more information of the booked clicked.

Built with: React, Typescript and SCSS

Fake-OS App

Recreating fake-os look of Windows 98. This app was created with HTML, Sass and vanilla Javascript. The styling was split with Sass into component specific files and were selected with BEM convention. The user will be able to view the current time and open Notepad app, Paint app and Address Book simultaneously. These apps will be able to be dragged around.

Built with: HTML, CSS/SCSS and Javascript

Advice App

This project fetches Data within the React App from a Firestore database. React router was used to create home, favourites and cart routes that users can go to. A grid of products are displayed and users are able to add it to their favourites or add the item to their cart. In the cart route users can add or remove items and perform a checkout which will mutate the item stock in the firestore database.

Built with: React, SCSS, React Router DOM and Firestore

Morse Code Translator

This project will take user input and translate it from latin to morse code or reverse depending on which mode it is on. Unit testing was used and done with jest to ensure functionality was sufficient. This project reinforced aspects of DOM manipulation and unit testing.

Built with: HTML, CSS/SCSS, Javascript and Jest
